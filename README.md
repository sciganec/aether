# 🌌 Quattuor Elementa (Aether)

> *"Світ складається з чотирьох першоелементів, і той, хто збагне їхній танок, керуватиме космосом."*

**Quattuor Elementa** — це глибока та атмосферна покрокова стратегічна гра, побудована на взаємодії чотирьох стихій (Вогню, Води, Землі та Повітря). Гра розгортається на полі 6х6, де гравці змагаються за контроль над дошкою за допомогою тактичних розміщень, комбінацій та передбачення ходів суперника.

🔗 **[Грати онлайн (GitHub Pages)](https://sciganec.github.io/aether)**

---

## ✨ Основні особливості

- **Унікальна механіка стихій:** Кожен елемент має свої сильні та слабкі сторони, вступаючи в "алхімічні реакції" з іншими фігурами на дошці.
- **Потужний Штучний Інтелект:** Супротивник (Голем) використовує сучасні алгоритми мислення для створення гідної конкуренції гравцеві.
- **Естетичний UI/UX:** Сучасний "Glassmorphism" дизайн, плавні анімації та преміальний зовнішній вигляд, що занурює в містичну атмосферу гри.
- **Адаптивний дизайн:** Грайте як на комп'ютері, так і на мобільному пристрої.

---

## 🧠 Технічна інформація (Під капотом)

Гра створена з використанням сучасних веб-технологій та складних алгоритмів пошуку:

* **Frontend:** React.js (v18), чистий CSS з використанням кастомних властивостей та сучасного позиціонування.
* **ШІ-Рушій (AI Engine):**
  * **Алгоритм Minimax з Alpha-Beta відсіченням:** Дозволяє ШІ "думати" на кілька кроків вперед, оптимізуючи свій вибір і відкидаючи свідомо програшні гілки розвитку.
  * **Ітеративний пошук у глибину (Iterative Deepening):** Робить мислення ШІ гнучким, дозволяючи йому адаптуватися до доступного часу на хід.
  * **Zobrist Hashing:** Високоефективна система кешування станів дошки. Завдяки запам'ятовуванню вже прорахованих комбінацій, ШІ працює блискавично швидко, не марнуючи ресурси на повторні обчислення.

## 🚀 Встановлення та запуск локально

Якщо ви хочете запустити гру на своєму комп'ютері або модифікувати код:

1. **Клонуйте репозиторій:**
   ```bash
   git clone https://github.com/sciganec/aether.git
   cd aether
   ```

2. **Встановіть залежності:**
   ```bash
   npm install
   ```

3. **Запустіть локальний сервер:**
   ```bash
   npm start
   ```
   Гра буде доступна у вашому браузері за адресою `http://localhost:3000`.

4. **Деплой на GitHub Pages (для розробників):**
   ```bash
   npm run deploy
   ```
